# Sample Output Reports - Daily Calorie Tracker

This folder contains sample output files generated by the **Daily Calorie Tracker CLI**.  
Each file represents a different scenario of daily calorie intake.

---

## Files

- **Harsh_sample_output.txt**  
  - Case: Exactly met daily limit  
  - Shows total calories exactly matching the calculated daily limit.

- **Rohit_sample_output.txt**  
  - Case: Exceeded daily limit  
  - Shows total calories exceeded the calculated daily limit.

- **Shweta_sample_output.txt**  
  - Case: Within daily limit  
  - Shows total calories consumed below the calculated daily limit.

---

## Notes

- Each file contains:  
  - Timestamp of the session  
  - Meal names and their calorie amounts  
  - Total & average calories  
  - Status compared to the daily calorie limit  

- All files are generated automatically by running the `tracker.py` script.
